Stashkit

Exposes MongoDB GridFS via a RESTful Interface

Features

  • Simple Installation
  • Administrative UI Console.
  • Create Users and Buckets to store files
  • File storage service which you can host in your own server
  • Token-based Authentication for Assets
  • POST data to the server to upload your files
  • Get Images/Video with query params to Manipulate/Transform them on the fly.
